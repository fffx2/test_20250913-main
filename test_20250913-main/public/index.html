WCAG 2.1 핵심 원칙

인식 가능 (Perceivable)
모든 사용자가 정보를 인식할 수 있어야 합니다.
운용 가능 (Operable)
모든 사용자가 인터페이스를 조작할 수 있어야 합니다.
이해 가능 (Understandable)
정보와 UI 조작 방법을 이해할 수 있어야 합니다.
견고함 (Robust)
다양한 기술로 접근할 수 있어야 합니다.
